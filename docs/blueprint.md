# **App Name**: banglarbhumigov.in

## Core Features:

- District, Block, and Mouza Selection: Dynamic population of dependent dropdown menus (District, Block, Mouza) to accurately narrow down geographical search parameters.
- Khatian and Plot Search Form: User interface for inputting Khatian or Plot numbers and selecting Khatian type (Normal, Lease, FHTD) to initiate information retrieval.
- Captcha Verification: Generation and validation of a numerical and alphabetical captcha to secure data access and prevent automated queries.
- Khatian/Plot Information Display: Retrieval and structured presentation of detailed land information (Khatian & Plot data) from a backend database based on user's query.
- Language Selection: Option to switch the user interface and potentially the retrieved information between 'Bengali' and other selected languages.
- Legal Terminology Explainer: An AI tool that provides on-demand, clear, and concise explanations for complex land-related terms, legal jargon, or abbreviations found within the retrieved Khatian and Plot information.
- Search History and Management: Store and display recent search queries to allow users to quickly re-access previously viewed Khatian or Plot details.

## Style Guidelines:

- Primary color: A deep, authoritative blue (#1C5EC8) to convey trust and official clarity. (HSL: 218, 75%, 45%)
- Background color: A subtle, light blue-grey (#F0F3F7) for a clean and uncluttered informational interface. (HSL: 218, 20%, 96%)
- Accent color: A warm, muted gold-orange (#CC8A20) to highlight calls-to-action and important information, ensuring contrast without being overly vibrant. (HSL: 30, 70%, 50%)
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable characteristics, suitable for detailed informational content.
- Utilize simple, functional, and universally recognizable line icons for form controls, navigation, and refresh actions, maintaining a professional and intuitive user experience.
- Adopt a clear two-column form layout for structured data entry and display, with thoughtful spacing to improve readability, mimicking the efficiency of official documents and supporting responsive design across devices.
- Implement subtle, non-intrusive loading indicators and transition effects for dropdown menu population, search results fetching, and form submissions to provide immediate user feedback without distraction.
